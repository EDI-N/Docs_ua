.. important:: Створити запит на анулювання документа неможливо, якщо документ знаходиться в стані:

    - відкликаний (в документа є квитанція №14)
    - відмовлений (в документа є квитанція №13)
    - запрошено анулювання (в документа є квитанція №11)

    ЄДРПОУ в підписі має відповідати ЄДРПОУ, зазначеному в документі (наприклад, дял COMDOC: *[КодКонтрагента]* або *[ІПН]* в одному з блоків *[ЕлектроннийДокумент/Сторони/Контрагент]*)
