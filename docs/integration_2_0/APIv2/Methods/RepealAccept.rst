**RESPONSE**

В тілі **відповіді** передаються:

**chain_uuid** - ідентифікатор ланцюжка, в якому знаходиться документ

**doc_uuid** - ідентифікатор документа, якому належить квитанція

**ticket_uuid** - ідентифікатор квитанції
