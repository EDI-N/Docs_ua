#############################################################
**Отримати обмеження контрагента за GLN**
#############################################################

Для роботи з цим методом користувач повинен бути `авторизованим <https://wiki.edin.ua/uk/latest/Distribution/EDIN_2_0/API_2_0/Methods/Authorization.html>`__.

.. note::
  Метод може використовуватись, як Продавцем (Виробником) так і Дистриб'ютором (Покупцем):

  * для отримання налаштованих обмежень з боку Продавця (Виробника) очікуються **gln** та **d_gln** параметри;
  * для отримання налаштованих обмежень з боку Дистриб'ютора (Покупця) очікуються **gln** та **producerId** параметри.
