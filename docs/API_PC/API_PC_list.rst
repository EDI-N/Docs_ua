Керування сповіщеннями
============================

Користувачі можуть налаштовувати лише власні сповіщення. Налаштування інших користувачів доступні Адміністраторам компанії (`ролі "Адміністратор" та "Супер адміністратор" <https://wiki.edin.ua/uk/latest/Personal_Cabinet/PCInstruction.html#user-roles>`__).

+-----------+----------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| **Метод** |                **URL запиту**                |                                                                   **Опис**                                                                    |
+===========+==============================================+===============================================================================================================================================+
| GET       | ``/api/oas/personal/notifications/settings`` | `Отримати налаштування сповіщень вказаного користувача <https://wiki.edin.ua/uk/latest/API_PC/Methods/GetNotificationsSettings.html>`__       |
+-----------+----------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| PUT       | ``/api/oas/personal/notifications/settings`` | `Встановити події, за якими НЕ потрібно надсилати сповіщення <https://wiki.edin.ua/uk/latest/API_PC/Methods/PutNotificationsSettings.html>`__ |
+-----------+----------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
| DELETE    | ``/api/oas/personal/notifications/settings`` | `Активувати всі налаштування сповіщень <https://wiki.edin.ua/uk/latest/API_PC/Methods/DelNotificationsSettings.html>`__                       |
+-----------+----------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------+
