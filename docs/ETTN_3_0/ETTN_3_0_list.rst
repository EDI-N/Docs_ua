🚛 Інструкції "ETTN" v3
################################

.. note::
   Шановні клієнти!

   З 02.09.2024 на боці Центральної бази даних (ЦБД) Міністерства розвитку громад та територій України (МІУ) спостерігаються технічні помилки, що можуть перешкоджати реєстрації електронних ТТН та коригуючих актів до ТТН. Нагадуємо, що на даний момент реєстрація ТТН в ЦБД є **необов'язковою**, тому ми тимчасово відключили автоматичну реєстрацію для більшості клієнтів. Основна робота з електронними ТТН залишається без змін — ви можете продовжувати використовувати наш сервіс для оформлення та обміну електронними ТТН між учасниками перевезення. Якщо для вас критично важлива інтеграція з ЦБД, будь ласка, зверніться до вашого відповідального менеджера з боку EDIN для налаштування індивідуального доступу.

   Дякуємо за розуміння!

   Команда EDIN
