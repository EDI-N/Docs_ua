.. role:: green

Схема і послідовність підписання документа "Акт коригування" в залежності від ролі Ініціатора:

1) "Вантажовідправник" **=>** :green:`"Вантажовідправник"` -> "Перевізник" -> "Вантажоотримувач" (3 підписанти)

2) "Перевізник" **=>** :green:`"Перевізник"` -> "Вантажовідправник" -> "Вантажоотримувач" (3 підписанти)

3) "Вантажоотримувач" **=>** :green:`"Вантажоотримувач"` -> "Вантажовідправник" -> "Перевізник" (3 підписанти)

4) "Замовник" **=>** :green:`"Замовник"` -> "Вантажовідправник" -> "Перевізник" -> "Вантажоотримувач" (4 підписанти)
